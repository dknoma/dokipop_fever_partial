# DokiPop! Fever
Portions of code I've written as '[Qats/kiyodev](https://kiyodev.itch.io/)' for DokiPop! Fever, a Visual Novel + Puzzle Game that won 7th out of 238 entries for [Doki Jam](https://itch.io/jam/doki-jam). While I can't show the entire codebase, I have been given permission to share the code that I have written for this project .

Play the full game over at https://alantellezs.itch.io/dokipop-fever! Here is the submission and results page if anyone is curious: https://itch.io/jam/doki-jam/rate/3803281
